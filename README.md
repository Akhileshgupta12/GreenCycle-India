# ♻ GreenCycle India

GreenCycle India is a waste recycling marketplace built using Django that connects waste sellers and buyers on a single platform.

## 🌐 Live Website

https://greencycle-india.onrender.com

## 🎥 Project Demo Video

https://youtu.be/A-QbgNWsqis

## 🚀 Features

* Seller Registration & Login
* Buyer Registration & Login
* Waste Upload System
* Waste Purchase System
* Notification System
* Seller Dashboard
* Buyer Dashboard
* Admin Panel
* Waste Management Platform

## 🛠 Technologies Used

* Python
* Django
* SQLite
* HTML
* CSS
* JavaScript
* Git
* GitHub
* Render
